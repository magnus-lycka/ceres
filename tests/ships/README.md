# Ship Test Cases

Ship test files in this directory serve as integration targets derived from
published or third-party ship stat blocks.

Before adding or modifying a ship test case:

- Read `docs/TEST_CASE_ASSEMBLIES.md` for conventions, the expected-values pattern,
  and how to document deviations from source material.
- Record every new ship in `docs/TEST_CASE_ASSEMBLIES.md` with its source and any
  relevant TCS or RI cross-references.

Ship test files should also be added to `test_gallery.py` to generate output
artefacts.
